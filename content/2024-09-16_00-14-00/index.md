## MagicPython拡張機能をインストールするべきか

vscodeでは、デフォルトでもPythonのハイライトがされているが、MagicPython拡張機能をインストールするべきか

## インストールしてみた

見た目が特に変わらない

## 何が違うのか調べてみたら公式に書いてあった

> **Attention VSCode users**: MagicPython is used as the default Python highlighter in Visual Studio Code. Don't install it unless you want or need the cutting edge version of it. You will likely see no difference because you're already using MagicPython.
>
> VSCode ユーザーの注意: MagicPython は Visual Studio Code のデフォルトの Python ハイライターとして使用されます。最新バージョンが必要でない限り、インストールしないでください。すでに MagicPython を使用しているため、違いはわからないでしょう。

[https://marketplace.visualstudio.com/items?itemName=magicstack.MagicPython#:~:text=Attention%20VSCode%20users%3A%20MagicPython%20is,you're%20already%20using%20MagicPython.](https://marketplace.visualstudio.com/items?itemName=magicstack.MagicPython#:~:text=Attention%20VSCode%20users%3A%20MagicPython%20is,you're%20already%20using%20MagicPython.)

## 結論

基本的にはインストール不要

vscodeの方の更新が間に合ってない時に限って、MagicPython拡張機能が必要になることもある

ということで、常に最新のPythonのバージョンを使ってるわけでもない私が困ることはないだろうと思い、アンインストールした

おわり。
